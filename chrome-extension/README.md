# レインズ照合ツール Chrome拡張機能

レインズの検索結果ページを開いた状態でボタンを押すだけで、照合アプリへ自動送信します。

## インストール手順

1. Chrome で `chrome://extensions/` を開く
2. 右上の「デベロッパーモード」をオンにする
3. 「パッケージ化されていない拡張機能を読み込む」をクリック
4. このフォルダ（`chrome-extension/`）を選択
5. 拡張機能が追加されたことを確認

## 初回設定

1. 拡張機能アイコンを右クリック → 「オプション」
2. **照合アプリのURL** を入力（例: `https://your-app.vercel.app`）
3. APIトークンを設定している場合は入力（任意）
4. 「保存」をクリック

## 使い方

1. レインズにログインして検索結果ページを開く
2. Chrome ツールバーの拡張機能アイコンをクリック
3. 「照合対象の物件」ドロップダウンから対象物件を選択
4. 「この検索結果を送信」をクリック
5. 照合スコアと結果が表示される
6. アプリの「照合リスト」画面でも結果を確認できる

## 複数ページの対応

レインズの検索結果が複数ページにわたる場合：

1. 1ページ目を開いて「この検索結果を送信」
2. 2ページ目に移動して再度「この検索結果を送信」（上書きされます）
3. より多くの候補を照合したい場合はページを絞り込んでから送信

> 現状は送信するたびに上書きされます。最後に送信したページの結果が残ります。

## アプリ側の設定（任意）

APIトークンで不正アクセスを防ぐ場合は、Vercel の環境変数に追加：

```
EXTENSION_TOKEN=任意の文字列
```

拡張機能のオプション画面で同じ文字列を入力してください。
